Bits of help:
Identical objects, should there be a different shader program for each or just one: 
 - Why don't you upload new uniforms for each object? You group all your objects that have the same shader-program together, then you cal glUseProgram 
for this program, and then for each object you upload the respective uniform data, and after you uploaded the uniform data for an object you can call glDraw. 
Then you upload the uniform data for the next object and call glDraw, and you don't need to call glUseProgram again because you are already using that program.
I think switching the shader-program for each object might become a bottleneck.
